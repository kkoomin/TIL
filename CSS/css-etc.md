# CSS ETC

## BEM

- Block Element Modifier

1. BLOCK: 해당 Component 자체로 의미 있는 Component
2. ELEMENT: BLOCK의 부분적 요소로 자체적으로 의미를 가지진 않음
3. MODIFIER: BLOCK이나 ELEMENT의 다른 버전

```
.block {}
.block__element {}
.block__element--modifier {}
```
